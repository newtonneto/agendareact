# Agenda
Agenda simples feita em ReactJS

## Requisitos
- [Yarn](https://yarnpkg.com/getting-started/install) 1.22.5

## Instalação
Após clonar o projeto, acesse a raíz de seu diretório e execute os seguintes comandos:
Baixar as dependências
```sh
  yarn
 ```
Executar o projeto
```sh
  yarn start
```
